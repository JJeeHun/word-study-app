# 통계 화면

---

## 구성

```
[오늘 현황]
  오늘 학습 N개  /  오늘 Done N개

[스트릭]
  연속 학습 N일

[전체 진행률]
  domain별 Done 비율 바 차트
  IT ████░░░░ 34%
  일상 ███████░ 60%

[날짜별 학습량 그래프]
  최근 N일 학습량 바 차트

[타이핑 / 객관식 정답률]
  타이핑 72%  /  객관식 88%

[자주 틀린 단어]
  단어 + 오답 횟수 상위 N개
```

---

## UX 규칙

- 비교 없음 — 남과의 비교 일절 없음
- 스트릭 경고/알림 없음 — 숫자 표시만
