# 온보딩 화면

> 앱 첫 실행 시 한 번만 표시.

---

## 흐름

```
1단계: 인트로 슬라이드 (현재 빈 페이지 placeholder — 추후 추가)
2단계: domain 관심사 선택 → domain 펫 랜덤 자동 배정
3단계: main 펫 선택 (종류 + 타입)
4단계: 시작 방식 선택
        [로그인 없이 시작]  [계정으로 시작]
```

---

## UX 규칙

- 각 단계는 스와이프로 이동
- 뒤로가기 가능 (이전 단계로)
- 모든 선택은 하단 영역에 배치
