# 세션 완료 화면

> SESSION_SIZE개 시험 완료 후 표시.

---

## 구성

```
펫 반응 (신남)

오늘 학습: N개
Done 전환: N개
오답 나중에 보기: N개

[한 번 더]   [홈으로]
```

---

## UX 규칙

- 버튼은 하단 배치
- 큐에 단어가 남아있으면 [한 번 더] 활성화
- 큐가 비었으면 "오늘 큐 완료!" 메시지 + 펫 특별 반응
