# 홈 화면

> 앱의 중심 허브.

---

## 구성

```
[오늘 현황]
  오늘 학습한 단어 수 / 오늘 Done 된 단어 수

[domain 진행률]
  IT 34%  /  일상 60%  / ...

[외우기 시작]   [시험 시작]    ← 원터치 진입, 하단 배치

[main 펫]  ← 우하단 플로팅 (위치 설정 가능)
```

---

## UX 규칙

- 네비게이션 바 없음 → main 펫 탭으로 이동
- 외우기 / 시험 버튼은 탭 즉시 진입 (추가 확인 없음)
