# 시험 모드 화면

---

## 진입 흐름

```
홈 [시험 시작] 탭
  → 모드 선택 바텀 시트
      타이핑 / 객관식
      세션 크기: [10] (기본값, 조절 가능)
      [시작]
  → 시험 진행
```

---

## 시험 화면 구성

**타이핑 모드**
```
큐 진행: 3 / 10

의미 표시 (크게)

[단어 입력 필드]   ← 하단, 키보드 위에 위치

[제출]
```

**객관식 모드**
```
큐 진행: 3 / 10

단어 표시 (크게)

① 의미 A
② 의미 B
③ 의미 C        ← 세로 4개, 우측 정렬 (엄지 존)
④ 의미 D
```

---

## 정답/오답 처리

**정답** → 펫 반응 → 자동으로 다음 문제

**오답**
```
[지금 보기]   [나중에 보기]
```
- 지금 보기 → 정답 표시, `answer_logs.reviewed_at = 현재 시각`
- 나중에 보기 → 오답 노트에 적재, `reviewed_at = NULL`
- 펫 "괜찮아, 같이 다시 해보자!"

---

## UX 규칙

- 객관식 보기는 세로 4개, 우측 정렬
- 타이핑 입력 필드는 키보드 바로 위 배치
- 오답 선택지([지금 보기][나중에 보기])는 하단 배치
