# 펫 시스템

> "나를 위한 앱" 철학을 표현하는 감정 동반자.
> 남과 비교 없음. 틀려도 격려. 오래 안 오면 그리워함.

---

## 펫 종류

| 타입 | 역할 | 표시 방식 |
|---|---|---|
| **main 펫** | 네비게이션 + 감정 표현 | 항상 글로벌 플로팅 (위치 설정 가능) |
| **domain 펫** | 해당 domain 단어 추천 | 추천 시에만 팝업으로 잠깐 등장 |

---

## main 펫

- 화면 어디서든 고정 표시 (4방향 위치 설정 가능)
- 탭하면 네비게이션 메뉴 오픈 (바텀 시트)
- 단어 추천 없음

**위치 설정** → `profiles.pet_position` (top-left / top-right / bottom-left / bottom-right, DEFAULT bottom-right)

**첫 사용자 힌트** → `profiles.has_seen_nav_hint = false`이면 "펫을 탭하면 메뉴가 열려요!" 툴팁 1회 표시

---

## 펫 상태 (main 펫)

프론트에서 학습 활동 기반으로 계산. 별도 저장 없음.

| 상태 | 조건 |
|---|---|
| 신남 | 오늘 학습함 |
| 보통 | 2~3일 경과 |
| 처짐 | 4~6일 경과 |
| 기운 없음 | 7일 이상 ("오랜만이야…") |

---

## 이벤트 반응

| 이벤트 | 반응 | 감정 |
|---|---|---|
| New 단어 등록 | "새로운 게 생겼어!" | 호기심·설렘 |
| 오답 | "괜찮아, 같이 다시 해보자!" | 격려·위로 |
| Done 달성 | "냠냠 맛있었어!" | 만족·배부름 |
| 7일 이상 미접속 | "오랜만이야…" | 그리움 |

---

## domain 펫

- domain 선택 시 랜덤 자동 배정
- 설정에서 변경 가능
- 해당 domain 미학습 단어 추천 시 팝업 등장
- "이 단어 외워볼래?" → 확인 or 무시 → 사라짐

---

## 온보딩 배정 흐름

```
domain 관심사 선택 → domain 펫 랜덤 자동 배정
main 펫 → 사용자가 직접 선택 (종류 + 타입)
```
