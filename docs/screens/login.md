# 로그인/회원가입 화면

> 선택적 진입. 홈·설정에서 접근.

---

## 레이아웃

```
[이메일 입력 필드          ]  [Google ]
[                          ]  [Kakao  ]
[이메일로 계속하기 버튼    ]

        로그인 없이 계속하기 (하단 텍스트 버튼)
```

이메일 입력은 좌측 넓게. 소셜 버튼은 우측 세로 배치 (엄지 닿는 위치).

---

## 로직

- 이메일 입력 후 → 계정 있으면 로그인 / 없으면 회원가입 자동 분기
- 재로그인 시 → 로컬 user 정보에서 email 자동 바인딩, 비밀번호만 입력
- Refresh Token 만료 시 이 화면으로 유도
